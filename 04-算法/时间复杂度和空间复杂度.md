1个数N除以多少次2到一
如果N是2次的K次幂如1024 2048
N/Math.pow(2, 0), N/Math.pow(2, 1),  N/Math.pow(2, 2), ..., N/Math.pow(2, k), 共k+1项 

Math.pow(2,k) = N   k=log2 N

T(n) = [log2 n] + 1 成立

如果N不是2的k次幂
Math.pow(2, k-1) < N < Math.pow(2, k) => k-1 < log2 N < k
